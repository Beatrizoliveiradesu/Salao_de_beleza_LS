# Studios Luz & Stylo - Website

Website moderno, responsivo e premium criado para o salão de beleza **Studios Luz & Stylo** (Manaus/AM).

## 🚀 Funcionalidades

- **Identidade Visual Premium:** Design dark mode sofisticado com detalhes em ouro champagne.
- **Pré-Agendamento via WhatsApp:** Formulário interativo que gera mensagens estruturadas para o WhatsApp do salão.
- **Filtro de Serviços:** Alternância rápida entre categorias de serviços (Cortes, Coloração, Unhas, Estética) sem recarregar a página.
- **Carrossel de Depoimentos:** Feedback dinâmico de clientes.
- **Scroll Reveal:** Animações sutis ao rolar a página.
- **Design Responsivo:** Adaptado perfeitamente a celulares, tablets e computadores.

## 🛠️ Tecnologias Utilizadas

- HTML5 Semântico
- CSS3 Customizado (CSS Grid, Flexbox, Variáveis CSS)
- JavaScript Nativo (ES6+, Intersection Observer API)

## 📦 Como Rodar o Projeto Localmente

1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. Clone este repositório ou baixe os arquivos.
3. Abra a pasta do projeto no terminal e execute o comando abaixo para iniciar o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
4. O navegador abrirá automaticamente com o site ativo em tempo real.
